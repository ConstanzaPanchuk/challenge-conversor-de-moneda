<h1 align="center"> Challenge Alura – Conversor de Monedas :currency_exchange: </h1>

:notebook: **Descripción del proyecto**

Proyecto desarrollado como challenge para el programa de Alura Latam, con el objetivo de aplicar los conocimientos adquiridos en los cursos de Java: aplicando la Orientación a Objetos, Java: trabajar con listas y colección de datos y Java: consumir API, escribir archivos y manejar errores.
El proyecto consiste en un conversor de monedas que permite al usuario interactuar a través de la consola, seleccionar distintas opciones de conversión y obtener el valor convertido utilizando tasas de cambio reales obtenidas desde una API externa.
Como mejora adicional, el proyecto puede ampliarse incorporando historial de conversiones, soporte para más monedas y registros con fecha y hora.

:hammer:**Funcionalidades del proyecto**

- Mostrar un menú interactivo en consola con opciones de conversión de monedas.
  
- Permitir al usuario ingresar el monto a convertir.

- Obtener las tasas de cambio actualizadas desde una API externa.
  
- Realizar conversiones entre distintas monedas (USD, ARS, BRL, COP, entre otras).
  
- Mostrar el resultado de la conversión con formato claro y decimales limitados.
  
- Validar opciones inválidas del menú y entradas incorrectas del usuario.
  
- Opción para salir del programa de forma segura.
